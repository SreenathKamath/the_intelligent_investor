# Intelligent Investor

Futuristic multi-page fintech website for budgeting, market intelligence, founder credibility, and investor learning.

## Files

- `index.html`
- `planner.html`
- `market.html`
- `learning.html`
- `screener.html`
- `founder.html`
- `styles.css`
- `app.js`
- `planner.js`
- `market.js`
- `knowledge.js`
- `learning.js`
- `screener.js`
- `ARCHITECTURE.md`

## Run

Open `index.html` in a browser, or run:

```bash
npm start
```

## Included features

- Hero section with founder branding for Sreenath S Kamath
- Separate home, planning, market, and founder pages
- Dedicated learning hub for fundamentals, technicals, and stock market education
- Dedicated screener for sectors, top weighted stocks, and stock search/detail
- Multi-input planning engine with budgeting ratios and goal feasibility checks
- Recommended portfolio mix and where-to-invest bucket guidance
- Live market index radar and chart proxy endpoints
- API-backed market news feed with free RSS integration
- Sector lens, screener workflows, knowledge hub, quizzes, external learning links, and founder trust page
- Architecture and API flow guidance for production scaling
